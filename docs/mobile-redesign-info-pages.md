# 정보 페이지 모바일 재설계 기획서

> `/about`, `/contact`, `/updates` — 컨텐츠 중심 페이지. 우선순위 낮지만 통일성을 위해 모바일 최적화 필요.

## 대상 페이지 (3개)

| # | 경로 | 페이지 | 주 컨텐츠 |
|---|---|---|---|
| 1 | `/about` | 사무실 소개 | 세무사 프로필, 비전, 리뷰, 오시는 길 |
| 2 | `/contact` | 연락처 / 위치 | 지도, 전화, 영업시간, 상담 폼 |
| 3 | `/updates` | 공지/업데이트 | 블로그 링크, 공지사항 리스트 |

## 페이지별 설계 방향

### `/about` — 소개
| 섹션 | 모바일 핵심 |
|---|---|
| Hero | "세무회계 새벽 소개" 한 줄 + 브랜드 이미지 |
| 세무사 프로필 | 고유빈 / 김근량 — 풀폭 카드 2장 세로 스택 |
| 비전/가치 | "왜 새벽을 선택하는가" 아이콘 리스트 |
| 실적 | 500건+, 15년+ 큰 숫자 그리드 |
| 리뷰 베스트 | 4~5개 선별, 스와이프 캐러셀 |
| 오시는 길 | 지도 + 주소 카드 (Location 컴포넌트 재사용) |
| CTA | 상담신청 / 전화 |

### `/contact` — 연락처
이 페이지는 **가장 전환 중요**. 심플하게 유지.

| 섹션 | 모바일 핵심 |
|---|---|
| Hero | "문의하기" 타이틀 |
| 연락 방법 카드 | **탭 시 즉시 동작**: 📞 전화 / 💬 톡톡 / ✉️ 이메일 (3장 세로 스택) |
| 위치 | 지도 + 주소 + "길찾기" 버튼 (네이버 지도 연동) |
| 영업 시간 | 요일별 깔끔한 테이블 |
| 주차 안내 | 간단한 2~3줄 |
| 상담 폼 | `/income-tax`의 ContactForm 재사용 |

### `/updates` — 공지/업데이트
| 섹션 | 모바일 핵심 |
|---|---|
| Hero | "공지사항" 타이틀 |
| 카테고리 탭 | 전체 / 공지 / 세무뉴스 / 블로그 |
| 포스트 리스트 | 날짜 + 제목 + 1줄 요약, 탭 시 외부 블로그로 이동 |
| Pagination | "더 보기" 버튼 (무한스크롤 X) |

## Variant 프롬프트 (페이지별)

**`/about`** — about/소개 페이지 모바일 설계. 세무사 프로필 2장 풀폭 카드 / 실적 숫자 그리드 / 리뷰 스와이프 / 오시는 길 지도.

**`/contact`** — 연락처 페이지. 전화·톡톡·이메일 3장 카드(탭 즉시 동작), 지도 카드, 영업시간, 상담 폼.

**`/updates`** — 공지/블로그 리스트. 카테고리 탭 + 포스트 리스트 (제목 + 날짜 + 1줄 요약).

## 작업 우선순위

```
1순위: /contact  ← 전환 경로상 중요
2순위: /about    ← 신뢰감 강화
3순위: /updates  ← 마지막 (공지 안 많음)
```
