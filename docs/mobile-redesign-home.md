## 섹션별 모바일 설계 방향

### Hero
- "세무회계 새벽" 브랜드 아이덴티티 + 한 줄 가치 제안
- **주력 CTA 2개**: [상담신청] 메인, [종소세 랜딩] 시즌 서브
- 신뢰 지표 (500건+, 2인 세무사, 15년+) 작은 뱃지로 CTA 하단
- 전체 높이는 1스크린 이내

### Services (4대 서비스)
- 상속세·증여세 / 양도소득세 / 개인사업자 기장 / 자금출처조사
- **2x2 그리드 카드**, 각 카드: 아이콘 + 서비스명 + 한 줄 설명 + 화살표
- 탭 시 상담 폼 스크롤 또는 블로그 관련 글로 이동

### Problem (공감 섹션)
- "이런 고민 있으신가요?" 질문형 3개
- 예: "양도세 신고하다 실수하면 어떡하지?" / "상속세 언제부터 준비해야 하지?" / "사업자등록 낸 김에 세금 처리 제대로 하고 싶은데..."
- 각 질문 → 답변 아코디언 또는 카드

### Testimonials
- 모바일은 **1장씩 스와이프**, dot 인디케이터
- 리뷰당 별점 + 짧은 요약 + 출처(네이버/카카오) 뱃지
- "더 많은 후기 보기" 버튼 (about 페이지 또는 네이버 플레이스 이동)

### CTA (전용 섹션)
- 풀블리드 그라데이션 배경
- 헤드라인: "세무 문제, 새벽에 맡기세요"
- 버튼 2개: [무료 상담 신청] / [📞 010-3262-3295]

### Location
- 지도 이미지(탭 시 네이버 지도 연동)
- 주소 / 전화 / 영업시간 / 주차안내를 1컬럼 리스트
- 위례서로 252 유원플러스송파 610호

## 벤치마크 참고

- **삼성전자 서비스센터 모바일** — 서비스 카드 그리드
- **토스** — CTA 섹션 스타일
- **올리브영** — 위치 정보 카드
- **쿠팡** — 하단 고정 CTA
