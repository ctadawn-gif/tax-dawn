## 수수료 체계

### 프리랜서(3.3%) 기준

| 수입금액 | 프리랜서 (사업자등록 X) |
|---|---|
| 단순경비율 대상자 | 100,000원 |
| 간편장부 연소득 3천만 원 이하 | 200,000원 |
| 간편장부 연소득 7천만 원 이하 | 300,000원 |
| 간편장부 연소득 7천만 원 초과 | 400,000원 |
| 복식부기의무자 | 별도 문의 |

### 사업자등록이 있는 경우
별도 상담 후 안내

### 안내사항 (필독)

1. 다수 사업장 소득, 근로소득, 기타소득 등 있으면 추가 비용 발생
2. 복식부기의무자로서 기장 미진행 시 소급기장 비용 별도
3. 프리랜서도 전년도 소득금액에 따라 복식부기의무자 해당 가능
4. **모든 금액 VAT 별도**
5. **4월 30일 이내 접수 + 입금 완료 시 2만 원 할인**

## 홈택스 수임동의 절차

1. 공동인증서/간편인증/ID·PW 로그인
2. 세무대리·납세관리 → 나의 세무대리 관리 → 나의 세무대리인 수임동의
3. 세무 대리인 정보 확인 후 동의하기
4. 기존 대리인 있는 경우: 해임 먼저 → 위 2·3 절차
